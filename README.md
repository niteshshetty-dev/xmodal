Application Requirements
Build XModal - a modal form that opens up upon clicking a button.
Also, ensure that proper data validation takes place upon clicking submit.

NOTE:

The Open Form button that appears on the initial render, must be created by using a button element and it must contain the text "Open Form".
The input element for email must have attribute id with "email" as the id. The input element for username must have attribute id with “username” as the id. The input element for Date of Birth must have attribute id with “dob” as the id. The input element for phone number must have attribute id with “phone” as the id.

The Submit button must have className attribute as "submit-button".

You must throw alert message "Invalid email. Please check your email address." if data validation fails for Email. You must throw alert message “**Invalid phone number. Please enter a 10-digit phone number.” if data validation fails for Phone Number. You must throw alert message “Invalid phone number. Please enter a 10-digit phone number.**” if data validation fails for Date of Birth.
